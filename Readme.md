# Apple Sales SQL Analysis

This project analyzes an Apple retail sales dataset using PostgreSQL.

## Dataset Tables
- Sales
- Products
- Stores
- Category
- Warranty

## Skills Used
- SELECT
- WHERE
- GROUP BY
- JOINS
- Aggregate Functions
- Date Functions
- CTEs
- Window Functions

## Project Overview
This repository contains solutions to 50+ business-oriented SQL questions using real-world retail sales data.
